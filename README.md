🔢 Number Guessing Game

This is a simple Number Guessing Game I created while learning JavaScript. It’s one of my first projects and helped me understand how basic logic works in web development.

📌 About the Project

In this game, the system generates a random number, and the user tries to guess it.
When the correct number is entered, it shows a success message.

This project focuses on basic interaction between the user and the webpage.

🚀 Features

Random number generation
User input to guess the number
Displays result when the guess is correct
Simple and clean interface

🛠️ Built With
HTML
CSS
JavaScript

💡 What I Learned

While building this project, I learned:

Handling user input
Writing simple logic
Understanding how events work
